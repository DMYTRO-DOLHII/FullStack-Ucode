HTML description

h1

Tags from h1 to h6 are used to define HTML headings.

<h1>Heading</h1>

Heading

p

The <p> tag defines a paragraph.

Browsers automatically add one empty line before and after each paragraph.

<p>Here is a paragraph</p>

Here is a paragraph

br

This tag represents a line break. It is an empty element, and does not need a closing tag.

form

The <form> tag is used to create an HTML form for user input.

The <form> element can contain one or more of the following form elements:

  1. <input>
  2. <label>
  3. <textarea>
  4. <button>
  5. <output>

CSS handbook

background-color

The background-color property sets the background color of an element. The background of an element is the total size of the element, including padding and borders (but not margins).

body {
background-color: rgb(55, 255, 109);
}

border

If the border-color is omitted, the color applied will be the text color.

.example {
border: 1px solid black;
}

padding

The padding of an element is the space between its content and its border.

  1. padding-top
  2. padding-right
  3. padding-bottom
  4. padding-left

.example {
padding: 50px;
}

margin

The margin property sets the spacing around an element, and is a shorthand property for the following individual margin properties:

  1. margin-top
  2. margin-right
  3. margin-bottom
  4. margin-left

.example{
margin: 50px;
}

Protocols

Description

A protocol is a set of rules and instructions for data transmission. The rules are defined for each step and process during communication between two or more computers. Networks must follow these rules to successfully transmit data.

HTTP/HTTPS

Hypertext Transfer Protocol Secure (HTTPS) is the secure version of HTTP, which is the primary protocol used to send data between web browsers and websites. HTTPS is encrypted to increase the security of data transmission. This is especially important when users transmit sensitive data, such as when logging into a bank account, email service, or insurance company.

DNS

DNS (Domain Name System) is a global system for converting IP addresses into user-friendly domain names. When a user tries to access a web address, for example, "example.com", their web browser or application performs a DNS query against a DNS server, providing the hostname. The DNS server receives the hostname and converts it into a numerical IP address, which the browser can then connect to.

IP

An Internet Protocol (IP) address is a numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication. An IP address serves two primary functions: host or network interface identification and location addressing.
